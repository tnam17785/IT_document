INTRODUCTION

Chương mở đầu giúp làm quen với lập trình thi đấu, các ngôn ngữ phổ biến, nhập xuất dữ liệu, toán học cơ bản và những kỹ năng cần thiết trước khi học thuật toán.

  • Ngôn ngữ
  • Input / Output
  • Làm việc với số
  • Rút gọn code
  • Toán học
  • Tài nguyên

1. Ngôn ngữ lập trình

Ba ngôn ngữ phổ biến nhất trong lập trình thi đấu là:

  • C++ – nhanh nhất, thư viện mạnh.
  • Python – cú pháp ngắn gọn, dễ học.
  • Java – ổn định và phổ biến.

Nếu muốn thi học sinh giỏi Tin hoặc Olympic Tin học, C++ gần như là lựa chọn bắt buộc. 

#include <bits/stdc++.h>
using namespace std;

int main(){
    return 0;
}

2. Input / Output

Mọi chương trình đều phải đọc dữ liệu đầu vào và xuất kết quả. Trong C++ thường sử dụng cin và cout. 

int a,b;
cin >> a >> b;

cout << a+b;

Để tăng tốc độ nhập xuất: 

ios::sync_with_stdio(false);
cin.tie(nullptr);

3. Làm việc với số

  • int: lưu số khoảng ±2 tỷ.
  • long long: lưu số rất lớn tới 10¹⁸.
  • double: số thực.
long long x = 1000000000000LL;

Trong các bài toán thuật toán, long long được sử dụng cực kỳ thường xuyên.

4. Rút gọn code

Lập trình viên thi đấu thường dùng typedef hoặc define để viết code nhanh hơn. 

typedef long long ll;

#define PB push_back
#define F first
#define S second

Điều này giúp code ngắn và tiết kiệm thời gian khi thi.

5. Toán học nền tảng

  • Tổng 1 + 2 + ... + n
  • Giai thừa n!
  • Dãy Fibonacci
  • Logarit
  • Lý thuyết tập hợp
  • Logic mệnh đề
1 + 2 + ... + n = n(n+1)/2

Toán học là nền tảng quan trọng nhất của thuật toán.
