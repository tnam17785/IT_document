TIME COMPLEXITY

Hiểu độ phức tạp thuật toán để viết code đủ nhanh.

  • Big O
  • Vòng lặp
  • Đệ quy
  • Các lớp
  • Ước lượng
  • Kadane

1. Big O

Big O mô tả tốc độ tăng thời gian chạy theo kích thước dữ liệu n.

O(1) O(log n) O(n) O(n log n) O(n²) O(n³) O(2ⁿ) O(n!)

2. Vòng lặp

for(int i=0;i<n;i++) { }

Độ phức tạp: O(n)

for(int i=0;i<n;i++) for(int j=0;j<n;j++) { }

Độ phức tạp: O(n²)

3. Đệ quy

void f(int n){ if(n==1) return; f(n-1); }

O(n)

void g(int n){ if(n==1) return; g(n-1); g(n-1); }

O(2ⁿ)

5. Ước lượng nhanh

n ≤ 20 -> O(2ⁿ) n ≤ 500 -> O(n³) n ≤ 5000 -> O(n²) n ≤ 10⁶ -> O(n log n)
